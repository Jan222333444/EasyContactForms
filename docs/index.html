Welcome to the Website of EasyContactForms

About

The EasyContactForms Backend utilizes a REST endpoint to receive and process form submissions. The endpoint expects a POST request with the necessary fields for the contact form entry. Once a request is received, it will either be stored in a relational database or forwarded to the specified email address. It is an Open Source Project so everyone can use and modify it for free. The application as well provides a Plugin API so everyone can modify the behavior.

Features

  • Redirect inquiries to your personal E-Mail Address without publishing it in the internet
  • Automatic resend of emails if delivery failed
  • Plugin interface to modify Application behavior
