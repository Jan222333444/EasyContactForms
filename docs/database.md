# Database structure

## Contact Form

- Table name: ``CONTACT_FORM``
- attributes: 
  - ``ID``: int autogenerated
  - ``EMAIL``: String
  - ``NAME``: String
  - ``SUBJECT``: String
  - ``MESSAGE``: String
  - ``EMAIL_SENT``: Boolean
  - ``CREATED``: Date
  - ``UPDATED``: Date
- Index:
  - ``ID``: Primary key
